Python API
==========

PyUCIS exposes two Python API styles:

* **Object-oriented API** (:doc:`oo-api`) — the primary Python interface,
  available for all backends (in-memory, SQLite, XML, YAML)
* **C-style API** (:doc:`c-style-api`) — a functional API that mirrors the
  Accellera UCIS C standard
